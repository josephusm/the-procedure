# The Procedure

A browser-based game about bureaucratic systems. You are a functionary. Cases arrive on your desk every day — anomalies, complaints, exceptions. You read them. You decide what to do with them.

You cannot refuse anything. The system does not have a "reject" option. It has *channels*.

[**Play it here →**](https://josephusm.github.io/the-procedure/)

---

This game was built to think through something I keep noticing: systems that do not suppress the exception — they route it. Every complaint has a correct form. Every dissent has an approved channel. The protest that can be filed and processed is already half-administered.

The mechanics are simple. The compliance score is not shown. The ending is routine.

---

## How to play

Turn on the machine. Read the case. Route it. The power button is in the bottom-right corner. Options are selected by clicking or by pressing the corresponding number key and Enter.

One sitting, about 15 minutes. No saves — like a shift.

## Running locally

Open `index.html` in a browser. That's it. No build step, no server required.
